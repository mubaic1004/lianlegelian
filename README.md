# 连了个连 🐰

连连看 × 消消乐的可爱风网页小游戏,机制向《羊了个羊》致敬:第 1 关热身教学,第 2 关地狱难度(实测通过率 ≈2%,且每一局保证有解)。

## 玩法

- **连线消除**:点两张相同且两折以内能连通的牌,直接消除。选中一张牌后,能连到的同伴会金色发光。连线飞在两张牌较高层之上,只会被不低于该层的牌挡住。
- **卡槽配对**:连不到的牌点一下进底部卡槽(有可连同伴的牌需要再点一下自己),槽内同类凑一对自动消除;塞入第 7 张单牌即失败。
- **道具**(每局各一次):撤回(也可在失败时当复活用)、洗牌、弹出(槽内 3 张回到牌面)。

## 本地运行

纯静态页面,任意 HTTP 服务器即可(ES Module 需要走 http):

```bash
python3 -m http.server 8931 --directory link-game
# 打开 http://localhost:8931
```

## 文件结构

- `engine.js` — 纯逻辑引擎(无 DOM,浏览器 / Node 通用):布局生成、遮挡判定、连连看 BFS 连通检测、卡槽状态机、难度校准 bot
- `ui.js` — 渲染与交互:选中/发光、连线动画、飞牌入槽、道具、结算弹窗、音效(WebAudio 合成)、彩带
- `style.css` — 黏土拟物(claymorphism)粉彩风格
- `sim.mjs` — 难度模拟器

## 难度是怎么校准的

第 2 关的每个布局种子都经过启发式 bot 批量试玩筛选:

```bash
node sim.mjs 2 1 70 300   # 关卡 2,种子 1~70,每个种子 300 局
```

入选标准:300 局中赢 1~12 局(通过率 0.3%~4%),**至少赢 1 局即确认该布局有解**。合格种子池写在 `engine.js` 的 `LEVELS[2].seeds`,每次开局随机抽一个。

调难度的旋钮(都在 `engine.js`):
- `LEVELS[2].types` — 图案种类数(越多越难,当前 20)
- 布局 `layout()` — 中央牌堆层数、左右盲堆深度(当前 7)、底部连连看排
- 改完后重新跑 `sim.mjs` 筛种子回填即可
